# WST website
Based on Joomla! 4.x CMS

### Requirements
You must have installed and available globally this CLI tools:
- php >= 8.0
- mysql/mariadb >= 10.3
- composer >= 2.1
- npm >= 6.14.6

### Build project
To build project run this file from project root directory (for Linux):
```
bin/build-project.sh
```
or (for Windows):
```
bin\build-project.cmd
```
